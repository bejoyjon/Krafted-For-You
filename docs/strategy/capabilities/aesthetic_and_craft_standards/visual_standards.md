# Visual Standards

| Element | Preferred | Avoid |
|---|---|---|
| Colors | Refined, warm, balanced | Too many bright clashing colors |
| Personalization | Subtle, meaningful | Oversized names everywhere |
| Packaging | Layered, protective, elegant | Random wrapping or visible clutter |
| Handmade finish | Clean edges, intentional texture | Rough unfinished look unless deliberate |
| Product mix | Coherent story | Random unrelated objects |
