# Customer Memory And Repeat Loop

Purpose: make customers feel remembered and create repeat/referral behavior.

## Fill Order

| Step | File | What to do |
|---:|---|---|
| 1 | [customer_memory_template.md](customer_memory_template.md) | Create memory rows for new and reconstructed customers where appropriate |
| 2 | [repeat_prompt_template.md](repeat_prompt_template.md) | Draft repeat occasion reminders |
| 3 | [referral_prompt_template.md](referral_prompt_template.md) | Draft referral prompts |
| 4 | [progress_and_feedback_loop.md](progress_and_feedback_loop.md) | Track follow-ups and outcomes |
| 5 | [success_metrics.md](success_metrics.md) | Review repeat/referral baseline |

## How To Analyze

Look for customers with repeated occasions, strong reactions, referrals, or workshop participation. Do not over-message; use memory to be thoughtful, not intrusive.

## Done For Version 1

Version 1 is ready when every new completed order has a customer memory row and permission status.
