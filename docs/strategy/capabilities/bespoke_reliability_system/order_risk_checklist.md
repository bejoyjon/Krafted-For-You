# Order Risk Checklist

Use before final quote and again before order confirmation.

| Risk | Yes/No | Action |
|---|---|---|
| Needed-by date under minimum lead time |  | Reshape or decline |
| Material availability unknown |  | Confirm before quote |
| Customer wants unlimited revisions |  | Set approval checkpoints |
| Vendor item not yet sourced |  | Add sourcing dependency |
| Delivery location uncertain |  | Confirm logistics |
| Budget does not match complexity |  | Reshape scope |
| Handmade technique is untested |  | Do not commit without trial buffer |

## How To Analyze

Every issue after confirmation should be traced back to the checklist. If a risk was not listed, add it.
