## Review Cadence

Weekly:

- count proposals using option paths,
- track selected options,
- review customer confusion,
- log founder edits.

Monthly:

- retire weak option paths,
- improve confusing labels,
- update budget and lead-time assumptions,
- add playbooks only when demand repeats.

## How To Alter Course

| Signal | What it means | Change |
|---|---|---|
| Customers ask for a full redesign | Options did not match intent | Improve brief or option logic |
| Customers cannot choose | Options are not differentiated enough | Clarify role of each option |
| Founder rewrites every option | Agent lacks brand/product context | Feed more reconstructed examples |
| Options create sourcing chaos | Architecture ignores feasibility | Add sourcing risk field earlier |
