# Occasion Playbook Template

Purpose: create reusable option paths for common occasions while preserving personalization.

## Playbook Template

| Occasion | Path | Emotional job | Typical inclusions | Personalization lever | Risk |
|---|---|---|---|---|---|
| Birthday | Delight | Make recipient feel celebrated | Treats, decor, small handmade item | Favorite colors/flavors | Becoming generic |
| Birthday | Identity | Show "this is so you" | Hobby-linked products, personal note | Recipient interest | Too much sourcing |
| Birthday | Keepsake | Create memory beyond the day | Handmade art/decor, message | Name/date/memory | Lead time |

## How To Fill

Start with birthday, anniversary, and farewell/gratitude. Add new playbooks only when repeated demand appears.

## How To Analyze

Review each playbook monthly:

- which paths converted,
- which paths created too much sourcing work,
- which personalization levers customers liked,
- which risks appeared repeatedly.
