## How To Analyze

1. Cluster reconstructed gifts by occasion, emotional intent, recipient type, handmade element, sourced element, complexity, and visual style.
2. Name repeated clusters as option paths.
3. Mark each path as safe, personal, or premium.
4. Test on new inquiries.
5. Retire options that confuse customers or create operational risk.

## Done For Version 1

This capability has a usable first version when:

- 20 reconstructed gift structures are analyzed,
- 5 reusable option clusters are named,
- 3 occasion playbooks are created,
- every qualified personal-gifting lead receives 2-3 option paths,
- option outcomes are reviewed monthly.
