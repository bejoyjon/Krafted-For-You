# Customer Emotion Brief Template

Use this for every serious inquiry and for reconstructed past orders.

## Template

| Field | Entry | Fill guidance |
|---|---|---|
| Brief ID |  | Use anonymous ID, e.g. `CEI-001` |
| Source |  | New inquiry, photo reconstruction, founder memory, referral |
| Occasion |  | Birthday, anniversary, farewell, new home, corporate, etc. |
| Buyer relationship to recipient |  | Wife to husband, friend to friend, founder to client, etc. |
| Recipient personality |  | Use free text first; map to archetype later |
| Desired feeling |  | What should the recipient feel? |
| Avoided feeling |  | What should the gift not feel like? |
| Personal clues |  | Hobbies, colors, favorite items, routines, memories |
| Emotional intensity |  | Low, medium, high |
| Gift story |  | One sentence explaining what the gift should communicate |
| Missing information |  | Budget, date, delivery, style, must-have items |
| Confidence |  | Low, medium, high |
| Founder correction |  | What the founder changed after review |

## How To Fill

If using past photos, fill only what is visible or remembered. Do not invent customer intent. Mark inferred details as inferred.

If using a new inquiry, fill this before generating options. If `desired feeling`, `budget`, or `needed-by date` is missing, ask follow-up questions before drafting detailed options.

## How To Analyze

Monthly, sort briefs by:

- occasion,
- relationship,
- desired feeling,
- recipient personality,
- missing information,
- founder corrections.

Repeated patterns become archetypes, occasion prompts, and option playbooks.
