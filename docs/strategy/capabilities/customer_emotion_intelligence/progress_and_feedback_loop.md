# Customer Emotion Intelligence Progress And Feedback Loop

Use this after every reconstructed order and every new serious inquiry.

## Review Cadence

Weekly:

- count briefs created,
- list missing fields,
- review founder corrections,
- identify confusing prompts.

Monthly:

- update archetypes,
- update occasion emotion map,
- update brief questions,
- update agent extraction prompt.

## How To Alter Course

| Signal | What it means | Change |
|---|---|---|
| Customers cannot answer a question | Prompt is too abstract | Reword with examples |
| Founder frequently corrects emotional intent | Agent/archetype is overreaching | Add confidence rules |
| Many briefs miss budget/date | Intake sequence is weak | Ask practical fields earlier |
| Option generation is generic | Brief lacks emotion or recipient detail | Add better emotional prompts |
